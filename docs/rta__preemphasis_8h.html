void rta_preemphasis_signal ( rta_real_t *  out_samples,
const rta_real_t *  in_samples,
const unsigned int  input_size,
rta_real_t *  previous_sample,
const rta_real_t  factor 
)

Apply preemphasis of 'factor' on 'in_samples' as 'out_sample'[0] = 'in_samples'[0] - 'factor' * (*'previous_sample') 'out_sample'[i] = 'in_samples'[i] - 'factor' * 'in_samples'[i-1], i>0

This calculation can not be in place: 'out_sample' != 'in_sample'

Parameters
out_samplessize is 'input_size'
in_samplessize is 'input_size'
input_sizeis the number of input and output samples and must be > 0
previous_sampleis updated as (*'previous_sample') = 'in_samples'['input_size'-1]
factoris generally 0.97 for voice analysis
