Downsample 'input' to 'output', by an integer factor, out of place. The 'output' samples are simple means of 'input' over 'factor' samples. The calculation can be in place if 'input' == 'output' .

Parameters
outputsize must be >= i_size / 'factor'
inputsize is 'i_size'
i_sizeis 'input' size
factormust be > 0

Cubic resampling of interleaved 'input' to 'output' by a factor, out of place.

Parameters
outputsize must be >= i_size / 'factor'
inputsize 'i_size' * 'i_channels'
i_sizeis 'input' number of sample frames
i_channelsis 'input' number of interleaved channels
factormust be > 0
Returns
1 if successful, 0 otherwise (in-place or input too short)
