Compute correlation between 'input_vector_a' and 'input_vector_b' into 'correlation'. If 'input_vector_a' == 'input_vector_b', it computes auto-correlation. This function can run in place if 'correlation' == 'input_vector_a' or 'correlation' == 'input_vector_b'.

\(C(i) = scale \sum_{f=0}^{max\_filter\_size-i} A(f+i) \cdot B(f), i=\{0,c\_size-1\}\)

This function is biased but uses all the information available for each coefficient.

Parameters
correlationsize is 'c_size'
c_sizeis the 'correlation' order + 1, 'c_size' must be > 0
input_vector_asize a_size must be >= 'max_filter_size'
input_vector_bsize b_size must be >= 'max_filter_size'
max_filter_sizeis the maximum shift for 'input_vector_a'. 'max_filter_size' must be > 'c_size'. In practice, 'max_filter_size' == a_size
scaleis a factor to multiply the 'correlation' values with
