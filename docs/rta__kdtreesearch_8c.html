int kdtree_search_knn ( kdtree_t *  t,
rta_real_t *  x,

const rta_real_t  r,
int  use_sigma,
kdtree_object_t *  y,
rta_real_t *  d 
)

Perform search in kd-tree structure t.

Return the k nearest neighbours in multi-dimensional data using the search tree t. The output vector d contains the squared distances to the r closest data vectors in kdtree_t::data according to the formula

\[ d_j = \sum_{j=0}^{ndim} \left( \frac{data_j - x_j}{\sigma} \right)^2 \]

Parameters
tkd-tree structure
xvector of kdtree_t::ndim elements to search nearest neighbours of
stridestride in vector x
kmax number of neighbours to find (actual number can be lower)
rmax squared distance of neighbours to find (r = 0 means no limit)
use_sigmause weights
youtput vector (size == r <= k) of (base, element) indices into original data kdtree_t::data
doutput vector (size == r <= k) of squared distances to data vectors
Returns
r = the number of actual neighbours found, 0 <= r <= k
