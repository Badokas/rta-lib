◆ rta_biquad_coefs()

void rta_biquad_coefs ( rta_real_t *  b,
rta_real_t *  a,
const rta_filter_t  type,
const rta_real_t  f0,
const rta_real_t  q,
const rta_real_t  gain 
)

Helper function calling the proper biquad coefficients calculation function, depending on the filter type.

Parameters
bis a vector of feed-forward coefficients
ais a vector of feed-backward coefficients
typecan be: 
  lowpass: H(s) = 1 / (s^2 + s/q + 1)
  highpass: H(s) = s^2 / (s^2 + s/q + 1)
  bandpass_cst_skirt: H(s) = s / (s^2 + s/q + 1)
       (The peak gain is q*gain)
  bandpass_cst_peak: H(s) = (s/q) / (s^2 + s/q + 1)
       (The peak gain is gain)
  notch: H(s) = (s^2 + 1) / (s^2 + s/q + 1)
  allpass: H(s) = (s^2 - s/q + 1) / (s^2 + s/q + 1)
  peaking: H(s) = (s^2 + s*(g/q) + 1) / (s^2 + s/(g*q) + 1),
       with g = sqrt(gain)
  lowshelf: H(s) = g * (s^2 + (sqrt(g)/q)*s + g)/
                        (g*s^2 + (sqrt(g)/q)*s + 1)
       with g = sqrt(gain)
  highshelf: H(s) = g * (g*s^2 + (sqrt(g)/q)*s + 1)/
                           (s^2 + (sqrt(g)/q)*s + g)
       with g = sqrt(gain)
f0is the cutoff frequency, normalised by the nyquist frequency.
qmust be > 0. and is generally >= 0.5 for audio filtering. q <= 1./sqrt(2.) is the limit for monotonic response for lowpass, highpass, lowshelf and highshelf types.
gainis linear and must be > 0.

Biquad computation, using a direct form I.

x           b0                                   y
--------+----->----->(  +  )-------->-----+------->
        |            ^ ^ ^ ^              |
        V   b1      / /   \ \        -a1  V
      [x-1]--->----/ /     \ ------<---[y-1]
        |           /       \             |
        V   b2     /         \       -a2  V
      [x-2]--->---/           -----<---[y-2]
Parameters

  

    
xis an input sample 

    
bis a vector of feed-forward coefficients. b0 is b[0], b1 is b[1] and b2 is b[2]. 

    
ais a vector of feed-backward coefficients. Note that a1 is a[0] and a2 is a1. 

    
statesis a vector of 4 elements: for an input 'x' and an output 'y', the states are, in that order, x(n-1), x(n-2), y(n-1), and y(n-2). Both can be initialised with 0. or the last computed values, which are updated by this function.

  

  




Returns
the output sample y
