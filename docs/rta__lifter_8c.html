◆ rta_lifter_weights()

int rta_lifter_weights ( rta_real_t *  weights_vector,
const unsigned int  cepstrum_order,
const rta_real_t  liftering_factor,
const rta_lifter_t  lifter_t,
const rta_lifter_mode_t  lifter_m 
)

Generate a vector of weights 'weights_vector' to lifter a cepstrum (usually to boost high coefficients). First coefficient is unchanged.

'factor' = exponent of x i^n liftering in exponential type (Auditory toolbox like) 'factor' = length of sin-curve liftering in sinusoidal mode (HTK style)

Parameters
weights_vectorsize is 'cepstrum_order'
cepstrum_ordernumber of input and output cepstrum coefficients
liftering_factor'lifter_t' == exponential_lifter: 0. is neutral, 0.6 is usual 'lifter_t' == sinusoidal_lifter: 'liftering_factor' must be >0.
  1. is neutral, 22. is usual
lifter_t'lifter_t' == exponential_lifter: Auditory Toolbox like 'lifter_t' == sinusoidal_lifter: HTK like
lifter_m'lifter_m' == rta_lifter_mode_normal: standard calculation, the one you want to 'lifter_m' == rta_lifter_mode_inverse: inverse calculation, to undo a liftering.
Returns
1 on success 0 on fail
