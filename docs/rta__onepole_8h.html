One-pole low-pass filter computed as: y(n) = f0 * x(n) - (f0 - 1) * y(n-1)

See also
rta_onepole_highpass
Parameters
xis an input sample
f0is the cutoff frequency, normalised by the nyquist frequency.
stateis the one sample delay state. It can be initialised with 0. or the last computed value, which is updated by this function.
Returns
the output sample y

One-pole high-pass filter computed as the difference between the input and a low-pass filtered input: y(n) = x(n) - ( f0 * x(n) - (f0 - 1) * y(n-1) )

See also
rta_onepole_lowpass
