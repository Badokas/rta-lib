Generate a matrix of weights 'weights_matrix' to combine a power spectrum into mel bands. As this matrix is rather sparse, generate also a bounds matrix 'weights_bounds', to avoid further null multiplications.

You can exactly duplicate the mel matrix in Slaney's mfcc.m as ret = rta_spectrum_to_mel_bands_weights(weights_matrix, weights_bounds, 512, 8000., 40, 133., 6855.5, 1., rta_hz_to_mel_slaney, rta_mel_to_hz_slaney, rta_slaney_mel);

Temporary memory allocation (and deallocation) is done inside this function.

rta_memory.h
