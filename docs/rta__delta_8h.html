Calculate the delta of 'input_vector' as 'delta' = 'input_vector' * 'weights_vector'

To calculate 'delta' over a ring buffer of 'input_vector', replicate the 'weights_vector' at the end (which memory size is now 2*'filter_size') and use (weights_vector + filter_size - 1 - current_input) as 'weights_vector' start pointer.

Parameters
deltais a scalar
input_vectorsize is 'filter_size'
weights_vectorsize is 'filter_size'
filter_sizeis the number of points to evaluate the delta

◆ rta_delta_normalization_factor()

rta_real_t rta_delta_normalization_factor ( const unsigned int  filter_size)

Generate a factor to multiply the delta with, in order to normalize the 'delta' values against the 'filter_size'. The delta is then the slope of the linear regression. Note that it is equivalent to multiply the 'weights_vector' values (but it is less precise, as errors in weights are added).

\(normalization = \frac{1}{2 \left(\sum\limits_{\theta=1}^{\theta=filter_size} \theta^2 \right)}\)

Parameters
filter_sizeis the number of points to evaluate the delta
Returns
normalization factor to multiply the delta with

◆ rta_delta_weights()

int rta_delta_weights ( rta_real_t *  weights_vector,
const unsigned int  filter_size 
)

Generate a vector of weights 'weights_vector' to calculate the delta (derivative) of a sequence of regularly sampled values. Use a 'filter_size'-points window. to calculate delta using a simple linear slope. This mirrors the delta calculation performed in feacalc etc. Delta corresponds to the mid-points values (which are not used, by the way).

'filter_size' must be odd and stricly positive.

'filter_size' == 7 is a common value to calculate the delta-mfcc and 'filter_size' == 5 is a common value for delta-delta-mfcc.

Note for HTK 'DELTAWINDOW' (same for 'ACCWINDOW'): 'filter_size' == ('DELTAWINDOW' * 2) + 1

Parameters
weights_vectorsize is 'filter_size'
filter_sizemust be odd and stricly positive.
Returns
1 on success 0 on fail
